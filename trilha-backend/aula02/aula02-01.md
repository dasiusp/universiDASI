## Tipos de variáveis
No mundo da programação temos tipos de variáveis. No curso de Introdução à Programação, vocês verão sobre os tipo em C, provavelmente. Em Javascript (JS) é um pouco diferente.
Em JS, não precisa especificar o tipo, ou seja, apenas declara a variável e ela pode assumir qualquer valor.
Nesta linguagem, temos os tipos:
### Number
Representa qualquer número! Nele, temos dois subtipos:
- **Integer:** para os íntimos, o inteiro (ou int), representa todos os inteiros.
- **Float:** chamando do bom português de ponto flutuante, representa os decimais.
E você declararia um inteitor como:
```javascript
let variavel1 = 10;
let variavel2 = 3.4;
```

### String
Uma string representa qualquer texto no programa, mas ela tem suas especificidade que serão abordados na aula de String.

```javascript
let string = "Um texto qualquer";
```

#### Concatenação
Concatenar é basicamente juntar uma string com outra string, número ou qualquer outro dado.
Há três de concatenar:
- **Adição:** Essa forma, junta dois dados com uma string, mas sem espaço.
    ```javascript
    let numero = 20;
    let string1 = "Um número "+numero; // "Um número 20"
    let string2 = "Texto anterior: "+string1; // "Texto anterior: Um número 20"
    let string3 = "Mais um texto "+"Outro texto";// "Mais um texto Outro texto"
    let string4 = numero+"Isto é um número"; // "20Isto é um número"
    ```
- **Vírgula:** Dessa forma, apenas funciona nos prints:
    ```javascript
    let numero1 = 20;
    let numero2 = "15";
    console.log(numero1,numero2);// "20 15"
    console.log("texto",numero,"texto"); // "texto 20 texto"
    ```
- **Crase:** A crase acaba sendo bastante popular, pois concatena já dentro da string, ficando mais legível.
    ```javascript
    let numero = 15;
    let string = `Número: ${numero}`; // "Número: 15"
    ```

### Boolean
Este tipo é o que indica o que é verdadeiro (true) e falso (false). Esse valor sempre é retornado depois de algum teste.
```javascript
let verdadeiro = true;
let falso = false;
```

### Undefined, Null e NaN
São tipos, ou melhor, valores que indicam alguma informação.
- undifined: indica que uma variável não tem nenhum valor valor, mas está declarada;
- null: indica que o objeto está vazio, não tem nada;
- NaN: indica que o objeto não é um número (NaN = Not a Number, não é um número).

Eles normalemnte aparecem em mensagens de erros.


## Converter tipos
Em alguns momentos você vai precisar mudar o tipo de sua variável para atender a determinada aplicação. Aqui irei listar as principais converções.

### Coverter para Number
Há três maneira de converter para um número:
- **Geral:** usando o Number(), é possível converter qualquer número, sendo geral;
    ```javascript
    let texto = "20";
    let numero = Number(texto);
    ```
- **Para inteiro:** caso queira que apenas um inteiro, usa-se Number.parseInt();
    ```javascript
    let texto = "204";
    let inteiro1 = Number.parseInt(texto); // 204
    let float = 15.5;
    let inteiro2 = Number.parseInt(float); // 15
    ```
- **Para float:** caso queira um float, usa-se Number.parseFloat();
    ```javascript
    let texto = "20.6";
    let float1 = Number.parseFloat(texto); // 20.6
    let inteiro = 40;
    let float2 = Number.parseFloat(inteiro); // 40.0
    ```

Caso não seja mandado um número válido, é retornado um NaN (Not a Number).

### Converter para String
É algo bem simples em converter para string e há duas formas de fazer isso.
- **"Gambiarra":** tem em relação à concatenação de variáveis com string. No Javascript, é possível usar dois acentos graves para delimitar strings. Explicaremos mais tarde sobre.
    ```javascript
    let numero = 10;
    let string = `${numero}`; // "10"
    ```
- **Palavra chave String:** usando a String(), converte para uma string.
    ```javascript
    let numero = 20.5;
    let string = String(numero); // "20.5"
    ```

### Converter para Boolean
Converter para Boolean é bem simples... Quase tudo é boolean! O quê?! Como assim??
Bem, o javascript interpreta os valores como Truthy ou Falsy.

Os Falsy seriam valores que são interpretados como falsos pelo JS, e eles são:
- false;
- 0, -0, 0n (este último sendo um bigint);
- "" ou '' (aspas vazias);
- null;
- undefined;
- NaN.

Os que não são Falsy, obviamente, são Truthy.

Com isso em mente, para converter um dado qualquer para um Boolean (true e false), usa-se o Boolean();
```javascript
let bool1 = Boolean(0); // false
let bool2 = Boolean("Texto"); // true
```

## Operadores aritméticos
Tá, temos agora os números, mas podemos fazer contas com eles? A resposta é sim, com os operadores aritméticos comuns, como adição(+), subtração(-), multiplicação(*) e divisão(/), além da potência (**). Mas aqui têm o operador especial: o `módulo (%)`. O que esse operador faz? Ele retorna o resto de uma divisão de inteiros!
Exemplo: 12 % 5 = 2.
