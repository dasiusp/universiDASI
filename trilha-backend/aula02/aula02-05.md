## Array e Objetos
### Array
Pense na seguinte situação: Há diversos carros de uma empresa, e você precisa guardar seus nomes. Aí você vai criando as variáveis:
```javascript
let carro1 = "HB20";
let carro2 = "Apolo";
let carro3 = "BMW X1";
let carro4 = "Celta";
let carro5 = "Corolla";
//...
```

Isso vai deixar várias variáveis criadas e toda vez que for ter um outro carro vai ter que criar uma nova variável. Mas se tivesse alguma forma de fazer uma lista, seria muito bom. E em Javascript há uma forma de fazer isso apenas usando colchetes!

Aqui vai um exemplo:
```javascript
let lista = []; // inicializado o array/lista/vetor
```

E dentro da lista pode colocar diversos elementos!
```javascript
let lista = ["Texto1", "Texto1", 20, 15.4];
```

E um array pode tem alguns métodos (funções específicas) que ajudam no desenvolvimento:
#### push() e pop()
O pop() remove o último elemento e o push() adiociona elementos na última posição.
```javascript
let mercado = ["Arroz", "Feijão", "Áçucar"];
mercado.pop();
console.log(mercado); // ["Arroz", "Feijão"]
mercado.push("Linguiça");
console.log(mercado); // ["Arroz", "Feijão", "Linguiça"]
```

#### unshift() e shift()
O unshift() adiciona elementos na primeira posição, enquanto o shift() remove.

```javascript
let mercado = ["Arroz", "Feijão", "Áçucar"];
mercado.shift();
console.log(mercado); // ["Feijão", "Áçucar"]
mercado.unshift("Carne moída", "Batata 1kg");
console.log(mercado); // ["Carne moída", "Batata 1kg", "Feijão", "Áçucar"]
```

#### Acessando elementos
Para acessar os elementos de um array usa-se colchetes e informa a posição (também chamado de indíce, ou index do inglês) do elemento, com a posição 0 é o começo de um array e assim por diante. Se tentar acessar uma posição negativa (tipo -1) ou uma posição que não exista um elemento é retorna um `undefined`.
```javascript
let array = ["Primeiro termo", "Segundo termo", "Terceiro termo"];
console.log(array[0]); // "Primeiro termo"
console.log(array[1]); // "Terceiro termo"
console.log(array[2]); // "Segundo termo"
```

#### Iteração
Com os arrays dá para fazer iteração com seus elementos. Mas o que é iteração? É passar por todos os elementos de um array.
Aqui vou mostrar três formas:
##### Laços (For, Do...While, While)
Usando a lógica de acessar elementos, é possível acesar cada um dos elementos usando um laço de repetição qualquer, controlando o seu índice.
Exemplo com o `FOR`:
```javascript
let array = [10,20,30,40];
let indice = 0;
for(indice; indice < array.length; indice++) console.log(array[indice]); 
```
- **array.length:** o `.length` é um método do array que devolve seu tamanho.

Por conta do tamanho do array ser o último indíce do vetor mais um, acaba que o laço termina antes de acessar qualquer ponto undefined.

Além disso, com o `for` dá para fazer a mesma coisa que o `if`, omitir as chaves quando o comando é apenas uma linha.

#### For...in
Para entender melhor, vamos mostrar a sintaxe:
```javascript
for(variável in objeto){
   doSomething();
}
```
Bem, o array é conhecido como um objeto, e todo o objeto tem suas propriedades (será mostrado mais para frente). As propriedades de um array são seus índices. O que o `for...in` faz é pegar essas propriedades e guarda na variável.
Exemplo:
```javascript
let redeSociais = ["Youtube", "Instagram", "TikTok", "Facebook", "Twitter"];
for(let variavel in redeSociais) 
    console.log(variavel); // 0, 1, 2, 3, 4
```

#### For...of
Este `for` já acessa os elementos e joga em uma variável, mas só funcionam em objetos iterável. Veja a sintaxe abaixo:
```javascript
let moedas = ["BRL", "GBP", "USD"];
let moeda;
for(moeda of moedas)
    console.log(moeda);

/*
Resultado:
BRL
GBP
USD
*/
```

### Objeto
Agora temos `let player1 = ["João", 400]`, em que o primeiro campo é o nome do jogador e o segundo é seus pontos. Assim, fica um pouco difícil interpretar isso, mas com um objeto é possível deixar isso mais claro.
**Declaração:**
```javascript
let obj1 = new Object();
// ou
let obj2 = {};
```
A partir desse ponto, é possível colocar os campos.

Exemplo 1:
```javascript
let player1 = new Object();
player1.name = "Maria";
player1.points = 400;
console.log(player1);
```

Exemplo 2:
```javascript
let player2 = {};
player2.name = "Willian B.";
player2.points = 1800;
console.log(player2);
```

Exemplo 3:
```javascript
let player3 = {
    name: "Fátima B.",
    points: 8000;
};
console.log(player3);
```

Isso tudo bate em relação à orientação à objeto. O `.nome` e `.points` são o que chamamos de propriedades. Com essa informação, é possível usar o `for...in` para acessar o nome de cada propriedade.
```javascript
let paciente = {
    nome: "Andreia",
    CPF: 12345678900,
    situacao: "Grave"
};
for(let props in paciente) console.log(`paciente.${props} = ${paciente[props]}`);
/* (é possível acesasr os elementos de um objeto usando o colchete com o nome da propriedade)

Resultados
paciente.nome: Andreia
paciente.CPF: 12345678900
paciente.situacao: Grave
*/
```

O `for...of` não funciona com o objeto, pois ele não é interável.

#### Construtor
É possível usar uma função como `construtor`, ou seja, usar a função para criar um objeto.
Exemplo 1:
```javascript
function Jogo(nome, tipo, preco) {
    this.nome = nome;
    this.modelo = tipo;
    this.preco = preco;
}

let HollowNight = new Jogo("HollowNight", ["Ação", "Aventura", "Exploração"], 60);
console.log(HollowNight);
/*
Resultado;
Jogo {
  nome: 'HollowNight',
  modelo: [ 'Ação', 'Aventura', 'Exploração' ],
  preco: 60
}
*/
```
Isso facilita para criar novos objetos. O `this` é algo bem complexinho e depende do cenário. No cenário acima, ela referencia ao objeto atual. Minha dica é pensar: "Quem esse this aponta?".
