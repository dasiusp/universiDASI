## Conhecendo o Visual Studio Code
Tá, é possível fazer tudo num bloco de notas, maaaaaaaaaaaaaaaaaaaaaaaaaaas é muito difícil (e feio). Então, o que podemos fazer? Bem, há um programa muito especial chamado Visual Studio Code, uma ferramenta muito usado no mercado e bem simples de usar.

### Instalação ([Página](https://code.visualstudio.com/download))
#### Linux
No site deles, há duas opções de instalação de um binário executável para linux, o .deb e .rpm. Para distro que aceitam as duas extensões, é só abaixar e executar oos arquivos.

Agora para baixar pelo binário já é um processo mais complexo.

- **Download do arquivo binário no site do Visual Studio Code**
    - Acessar o site do Visual Studio Code e baixar o .tar.gz: https://code.visualstudio.com/Download
    - Ou baixar pelo terminal
        ```bash
        wget https://update.code.visualstudio.com/latest/linux-x64/stable -O vscode.tar.gz
        ```

- **Extrair o arquivo**
    - `tar -xzf code-stable-x64-1770920270.tar.gz`, troca o nome antes do .tar.gz pela arquivo baixado e executar no terminal;
    - Entrar na pasta criada por essa extração.

- **Executar o VS Code**
    - Dentro da pasta, executar o `./code` no terminal e o VS Code abrirá;
    - Ele só executa na pasta, mas vamos mudar isso!

- **Mover para /opt**
    - `/opt` é o diretório padrão para instalações manuais;
    - `sudo mv VSCode-linux-x64 /opt/vscode`
    - Agora pode-se escrever `/opt/vscode/code` no terminal e abrirá o VS Code

- **Criar o comando global "code"**
    - Digitar `/opt/vscode/code` é bem complicado, vamos resumir isso para `code`;
    - No terminal: `sudo ln -s /opt/vscode/code /usr/local/bin/code`;
    - Aí é possível usar o `code` no terminal e executa o programa.

- **Criar atalho no menu**
    - Mas se o terminal te deixa com medo, dá para colocar ele no menu do linux;
    - Cria-se um arquivo usando o terminal `sudo nano /usr/share/applications/vscode.desktop`;
    - Vai abrir uma tela no terminal, isso é um editor de texto;
    - Agora cola o texto abaixo:
        ```ini
        # Indica que o arquivo descreve uma entrada de aplicativo
        # É obrigatório. Sem isso, o sistema não reconhece o arquivo como válido
        [Desktop Entry]
        # Nome que aparecerá no menu
        Name=Visual Studio Code
        # Descrição curta do aplicativo que aparece quando passa
        # o mouse está em cima programa. Não é obrigatório.
        Comment=Code Editing. Redefined.
        # Comando que será executado quando você clicar no ícone.
        Exec=/opt/vscode/code
        # Define o ícone do menu
        Icon=/opt/vscode/resources/app/resources/linux/code.png
        # Define se o programa precisa abrir dentro de um terminal.
        #Quando false, só abre o aplicativo; quando true, abre um terminalk junto
        Terminal=false
        # Define o tipo da entrada.
        # Application -> programa normal
        # Link -> atalho para site
        # Directory -> diretório especial
        Type=Application
        # Define em qual categoria do menu o app aparecerá.
        # Há algumas categorias, como Development, Utility, Network, Graphics, Office, System, Game. 
        # Eles são separados por ponto e vírgula (;) e a linha termina com `;`.
        Categories=Development;IDE;
        ```
    - Agora ao usar o Menu, é possível acessar o VS Code.
